Smart Agriculture Monitoring System

Final Year Project Portfolio

Project Overview

This project aims to create a smart agriculture monitoring system using IoT sensors to track soil moisture, temperature, and humidity in real time. The data is sent to a cloud server and displayed via a user-friendly dashboard.

Technologies Used

  • HTML, CSS, JavaScript
  • Arduino UNO
  • ESP8266 Wi-Fi Module
  • Firebase Realtime Database
